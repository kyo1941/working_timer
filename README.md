# Working Timer
## 概要
- 労働時間の計測と給料の計算を行うアプリ
- タイマー → 労働時間の記録 → 指定範囲の給料の計算 の流れをスムーズに行う

## 使用技術
Kotlin，Jetpack Compose，Room，Kotlin coroutines
